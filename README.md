# Calorine Velas

E-commerce de velas artesanais com vitrine, carrinho persistente, checkout, login com JWT e painel administrativo.

## Rodar

Use o perfil dev padrao:

```bash
mvn spring-boot:run
```

Admin de teste:

- E-mail: `admin@calorine.com`
- Senha: `admin123`

## URLs

- Loja: `http://localhost:8080/`
- Admin: `http://localhost:8080/#admin`
- H2 Console: `http://localhost:8080/h2-console`

JDBC dev: `jdbc:h2:file:./data/calorine`
